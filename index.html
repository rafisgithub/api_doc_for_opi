API and Database change Documentation

Add User Interaction

  • Endpoint: POST /user_interaction
  • Method: POST
  • Purpose: This endpoint allows you to add user interactions.

Match Users

  • Endpoint: match_users
  • Method: POST
  • Purpose: Use this endpoint to match users.

Start a Chat Session

  • Endpoint: chatting_started
  • Method: POST
  • Purpose: Initiates a chat session between users.

Search Universities

  • Endpoint: universities/{name?}
  • Method: GET
  • Purpose: Retrieve all university Name. The optional name parameter can be used for filtering by university name. and search option

Unmatch a User

  • Endpoint: unmatched_user
  • Method: POST
  • Purpose: This endpoint allows you to unmatch a user.
